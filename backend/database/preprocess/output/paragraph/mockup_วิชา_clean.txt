# คำอธิบายรายวิชา สาขาปัญญาประดิษฐ์และวิทยาการข้อมูล


## AIE213 - Computer Programming
หน่วยกิต: 3 (2 – 3 – 5)  
หมวด: วิชาแกน  
กลุ่ม: วิศวกรรมพื้นฐาน  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการเขียนโปรแกรมคอมพิวเตอร์เบื้องต้น ครอบคลุมโครงสร้างข้อมูลพื้นฐาน การใช้ตัวแปร อาเรย์ ฟังก์ชัน และการเรียกซ้ำ ฝึกการออกแบบอัลกอริทึมเพื่อแก้ปัญหาอย่างเป็นระบบ รวมถึงการเขียนโปรแกรมเชื่อมต่อกับฮาร์ดแวร์เบื้องต้นและการใช้ไลบรารีมาตรฐานสำหรับงานประยุกต์

### Course Description (English)
Study of fundamental programming concepts including variables, arrays, functions, recursion, and basic data structures. Emphasis on algorithm design, problem solving, hardware interaction, and usage of standard libraries for practical applications.

---

## AIE223 - Artificial Intelligence
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเฉพาะด้านบังคับ  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาพื้นฐานของปัญญาประดิษฐ์ ครอบคลุมแนวคิดการเรียนรู้ของเครื่อง ระบบผู้เชี่ยวชาญ โครงข่ายประสาทเทียม และอัลกอริทึมการค้นหา วิเคราะห์การประยุกต์ใช้ปัญญาประดิษฐ์ในงานด้านการรู้จำภาพ การประมวลผลภาษาธรรมชาติ และระบบอัตโนมัติในอุตสาหกรรม

### Course Description (English)
Introduction to artificial intelligence concepts including machine learning, expert systems, neural networks, and search algorithms. Applications in image recognition, natural language processing, and industrial automation are explored.

---

## AIE311 - Data Structure and Algorithm
หน่วยกิต: 3 (2 – 3 – 5)  
หมวด: วิชาเฉพาะด้านบังคับ  
วิชาบังคับก่อน: AIE213  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการโครงสร้างข้อมูล เช่น ลิสต์ สแตก คิว ต้นไม้ และกราฟ รวมถึงอัลกอริทึมสำหรับการค้นหาและการจัดเรียงข้อมูล วิเคราะห์ประสิทธิภาพของอัลกอริทึมด้วยแนวคิดเชิงเวลาและหน่วยความจำ ฝึกการประยุกต์ใช้โครงสร้างข้อมูลในการแก้ปัญหาทางวิศวกรรม

### Course Description (English)
Study of fundamental data structures such as lists, stacks, queues, trees, and graphs. Covers searching and sorting algorithms, performance analysis in terms of time and memory, and practical problem-solving applications.

---

## AIE313 - Database System
หน่วยกิต: 3 (2 – 3 – 5)  
หมวด: วิชาเฉพาะด้านบังคับ  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการออกแบบฐานข้อมูลเชิงสัมพันธ์ การสร้างตาราง ความสัมพันธ์ และข้อจำกัดข้อมูล ฝึกการใช้งานภาษา SQL สำหรับการจัดการข้อมูล วิเคราะห์แนวคิดการจัดเก็บข้อมูลขนาดใหญ่ ความปลอดภัย และการสำรองข้อมูลในระบบฐานข้อมูลสมัยใหม่

### Course Description (English)
Principles of relational database design including tables, relationships, constraints, and SQL for data manipulation. Topics include data storage, security, backup, and modern database management techniques.

---

## AIE321 - Big Data Processing and Analytics
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเฉพาะด้านบังคับ  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการจัดการข้อมูลขนาดใหญ่ การประมวลผลแบบกระจาย การใช้เครื่องมือสำหรับ Big Data เช่น Hadoop และ Spark วิเคราะห์ข้อมูลเชิงสถิติและเชิงพยากรณ์เพื่อสนับสนุนการตัดสินใจ รวมถึงการประยุกต์ใช้ในภาคธุรกิจและอุตสาหกรรม

### Course Description (English)
Study of big data management and distributed processing systems such as Hadoop and Spark. Covers statistical analysis, predictive analytics, and real-world applications in business and industry.

---

## AIE322 - Supervised Machine Learning
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเฉพาะด้านบังคับ  
วิชาบังคับก่อน: IE213  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาวิธีการเรียนรู้ของเครื่องแบบมีผู้สอน เช่น การถดถอยเชิงเส้น การจำแนกข้อมูลด้วยอัลกอริทึมต่าง ๆ การวัดประสิทธิภาพโมเดล และเทคนิคการปรับปรุงความแม่นยำ วิเคราะห์ปัญหาการเกิด overfitting และ underfitting พร้อมฝึกประยุกต์ใช้กับชุดข้อมูลจริง

### Course Description (English)
Study of supervised machine learning techniques including linear regression, classification algorithms, model evaluation, and performance improvement methods. Covers overfitting, underfitting, and real dataset applications.

---

## AIE331 - Computer Vision
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเลือกเฉพาะด้าน  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการประมวลผลภาพดิจิทัล การตรวจจับวัตถุ การรู้จำรูปแบบ และการประยุกต์ใช้โครงข่ายประสาทเชิงลึกในงานด้านการมองเห็นของคอมพิวเตอร์ ฝึกพัฒนาโปรแกรมเพื่อวิเคราะห์ภาพและวิดีโอในสถานการณ์จริง

### Course Description (English)
Principles of digital image processing, object detection, pattern recognition, and deep learning applications for computer vision. Practical development for image and video analysis is emphasized.

## AIE341 - Natural Language Processing
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเลือกเฉพาะด้าน  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาหลักการประมวลผลภาษาธรรมชาติ การวิเคราะห์ข้อความ การสร้างแบบจำลองภาษา และการประยุกต์ใช้เทคนิคการเรียนรู้ของเครื่องในการแปลภาษา การสรุปข้อความ และระบบสนทนาอัตโนมัติ

### Course Description (English)
Study of natural language processing techniques including text analysis, language modeling, and machine learning applications for translation, summarization, and conversational systems.

## AIE342 - Deep Learning
หน่วยกิต: 3 (3 – 0 – 6)  
หมวด: วิชาเลือกเฉพาะด้าน  
วิชาบังคับก่อน: AIE322  

### คำอธิบายรายวิชา (ภาษาไทย)
ศึกษาสถาปัตยกรรมโครงข่ายประสาทเชิงลึก เช่น CNN, RNN และ Transformer วิเคราะห์กระบวนการฝึกโมเดลเชิงลึกและการปรับพารามิเตอร์ ฝึกการประยุกต์ใช้ deep learning กับข้อมูลภาพ เสียง และข้อความ

### Course Description (English)
Study of deep neural network architectures such as CNNs, RNNs, and Transformers. Covers training processes, parameter tuning, and applications to image, audio, and text data.
